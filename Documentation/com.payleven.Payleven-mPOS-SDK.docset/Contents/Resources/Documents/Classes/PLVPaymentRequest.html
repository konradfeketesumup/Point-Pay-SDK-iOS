Payment identifier.

=======

Mandatory unique identifier (e.g order number) of the payment request. The ID can be any set of alphanumeric characters [A-Z, a-z, 0-9]. This payment identifier must be included in receipts and will be reflected on payleven’s receipt images. Please retain this identifier for your records and for further reference (e.g Refund).
